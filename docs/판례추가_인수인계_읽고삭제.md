# 📌 [인수인계 · 읽고 나면 삭제] 도시정비 판례 64건 corpus 추가 요청

> **다음 Claude에게:** 이 파일은 임시 인수인계 메모다. 아래 판례들을 `corpus/precedents/`에
> 추가하는 작업을 마치면 **이 파일(`docs/판례추가_인수인계_읽고삭제.md`)을 삭제**하고 커밋해라.
> 목적을 다한 일회용 문서이므로 저장소에 남기지 말 것.

---

## 배경 (왜 이 목록이 만들어졌나)

- 기존 판례 수집 스크립트 `scripts/fetch-precedents.py` 는 국가법령정보센터 판례를
  **제목(사건명) 검색(`search=1`)만** 사용해서, 정비 쟁점 판례를 상당수 놓치고 있었다.
  (판례 사건명은 "관리처분계획취소" 같은 소송물 이름이라 단지명·쟁점어로는 안 잡힘)
- 이번에 **본문 전문 검색(`search=2`)** 까지 병행해 도시정비 판례를 전수 훑고,
  **대법원 + 판결요지 존재 + 참조조문에 「도시 및 주거환경정비법」 포함** 조건으로 필터링,
  기존 corpus 12건과 중복 제거한 결과가 아래 **신규 64건**이다.
- 오탐(환지청산금·토지구획정리·도시개발법·자동차정비·취득세 등)과 형사(뇌물)는 이미 제외했다.

## ⚠️ 한계 (미리 알아둘 것)

- **신반포2차 상가 산정비율(0.1) 판례(2심 서울고법 2025나205452, 대법원 심리불속행)는
  국가법령정보센터에 미등재라 이 목록에 없다.** 심리불속행 기각은 이유 없는 결정이라 판례DB에
  안 실리고, 2심 최신 하급심은 반영이 느리다. 필요하면 판결서 인터넷열람으로 수동 확보해야 한다.
- 무료 공공 판례 API는 국가법령정보센터(=공공데이터포털 법제처 판례)가 사실상 유일하다.
  최신 하급심은 케이스노트·엘박스·빅케이스 등 상용 서비스에만 있고 공식 오픈API가 없다.

## 작업 방법

`scripts/fetch-precedents.py` 의 `CASES` 리스트를 아래 블록으로 교체(또는 이어붙임)하고 실행:

```bash
python3 scripts/fetch-precedents.py
```
→ 각 판례ID를 상세조회(`target=prec`)해서 `corpus/precedents/{사건번호}.md` 를 생성한다.
(스크립트가 판시사항·판결요지·참조조문·참조판례까지 프론트매터 포함 md로 변환)

실행 후: 생성된 md들 확인 → `scripts/build-index.mjs` 등 인덱스 재빌드가 필요하면 함께 수행 →
**이 인수인계 파일 삭제** → 커밋.
